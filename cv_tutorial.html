üìö What is Computer Vision?

Computer vision is teaching computers to "see" and understand images and videos, just like humans do. Think of it as giving your computer eyes and a brain to recognize things!

Real-World Examples:

  • Security Apps: Detect if someone is at your door or recognize faces
  • E-commerce Apps: Search products by taking photos, or try on clothes virtually
  • Social Media: Auto-tag friends in photos
  • Healthcare: Detect diseases from medical scans

Step 4: Building an Object Detection System

Now let's detect objects in images - this is the exciting part!

1 Understanding Object Detection

Object detection finds objects in images and draws boxes around them. For example, finding all the people, cars, or products in a photo.

2 Using a Pre-trained Model

Instead of training AI from scratch (which takes weeks!), we'll use a model that's already smart. Create object_detector.py:

import cv2 import numpy as np # Load YOLO (You Only Look Once) - a popular object detector # Download these files from the YOLO website first net = cv2.dnn.readNet("yolov3.weights", "yolov3.cfg") # Load class names (things the AI can recognize) with open("coco.names", "r") as f: classes = [line.strip() for line in f.readlines()] # Load an image image = cv2.imread("test.jpg") height, width = image.shape[:2] # Prepare the image for YOLO blob = cv2.dnn.blobFromImage(image, 1/255, (416, 416), swapRB=True, crop=False) # Feed the image to the neural network net.setInput(blob) output_layers = net.getUnconnectedOutLayersNames() detections = net.forward(output_layers) # Process detections for detection in detections: for obj in detection: scores = obj[5:] class_id = np.argmax(scores) confidence = scores[class_id] if confidence > 0.5: # Only show if we're 50%+ confident # Get box coordinates center_x = int(obj[0] * width) center_y = int(obj[1] * height) w = int(obj[2] * width) h = int(obj[3] * height) x = int(center_x - w/2) y = int(center_y - h/2) # Draw rectangle and label cv2.rectangle(image, (x, y), (x+w, y+h), (0, 255, 0), 2) label = f"{classes[class_id]}: {confidence:.2f}" cv2.putText(image, label, (x, y-10), cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0,255,0), 2) # Show result cv2.imshow("Object Detection", image) cv2.waitKey(0) cv2.destroyAllWindows()

  • yolov3.weights (model file)
  • yolov3.cfg (configuration)
  • coco.names (list of objects it can detect)
Search "YOLOv3 weights download" to get these free files.

Step 5: Building Real Applications

üõ°Ô∏è For Security Apps:

Face Detection System import cv2 # Load face detector (comes with OpenCV!) face_cascade = cv2.CascadeClassifier( cv2.data.haarcascades + 'haarcascade_frontalface_default.xml') # Open webcam cap = cv2.VideoCapture(0) while True: ret, frame = cap.read() gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY) # Detect faces faces = face_cascade.detectMultiScale(gray, 1.1, 4) # Draw rectangles around faces for (x, y, w, h) in faces: cv2.rectangle(frame, (x, y), (x+w, y+h), (255, 0, 0), 2) cv2.putText(frame, "Face Detected", (x, y-10), cv2.FONT_HERSHEY_SIMPLEX, 0.9, (255,0,0), 2) cv2.imshow('Security Camera', frame) # Press 'q' to quit if cv2.waitKey(1) & 0xFF == ord('q'): break cap.release() cv2.destroyAllWindows()

What this does: Opens your webcam, detects faces in real-time, and draws boxes around them. Perfect for security systems!

üõí For E-commerce Apps:

Product Image Classifier

Train a simple classifier to recognize products:

from tensorflow import keras import numpy as np # Load a pre-trained model (like MobileNet) model = keras.applications.MobileNetV2(weights='imagenet') # Prepare your image image = keras.preprocessing.image.load_img('product.jpg', target_size=(224, 224)) image_array = keras.preprocessing.image.img_to_array(image) image_array = np.expand_dims(image_array, axis=0) image_array = keras.applications.mobilenet_v2.preprocess_input(image_array) # Predict what it is predictions = model.predict(image_array) decoded = keras.applications.mobilenet_v2.decode_predictions(predictions, top=3) # Show results print("This product looks like:") for i, (imagenet_id, label, score) in enumerate(decoded[0]): print(f"{i+1}. {label}: {score*100:.2f}% confident")

Use case: Users upload a photo, and your app identifies the product automatically!

Step 6: Making Your System Better

1. Collect More Data

The more images your AI sees, the smarter it gets. Aim for at least 100 images per category you want to detect.

2. Clean Your Data

  • Remove blurry or low-quality images
  • Make sure images are labeled correctly
  • Use diverse images (different angles, lighting, backgrounds)

3. Try Different Models

Different models work better for different tasks:

  • YOLO: Fast, great for real-time detection
  • ResNet: Very accurate for classification
  • MobileNet: Works on phones and slower computers

4. Test, Test, Test!

Always test your system with new images it hasn't seen before to make sure it works in real situations.

üîß Common Problems & Solutions

Problem: "Module not found" error

Solution: Reinstall the library: pip install library_name --upgrade

Problem: Code is too slow

Solutions:

  • Use smaller images (resize to 416x416 or 224x224)
  • Use lighter models like MobileNet or YOLOv3-tiny
  • Process every other frame instead of every frame

Problem: Inaccurate detections

Solutions:

  • Adjust confidence threshold (try 0.3 to 0.7)
  • Use better lighting in your images
  • Train your own model with specific data

Problem: Can't open camera

Solution: Change cv2.VideoCapture(0) to cv2.VideoCapture(1) or check camera permissions.
